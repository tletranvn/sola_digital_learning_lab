# Matériel Moodle historique

Ce dossier regroupe les artefacts du premier prototype Moodle :

- `contenu-a-coller.html`, ancienne source de contenu préparée pour l'éditeur Moodle ;
- captures locales `moodle_content1.png` et `moodle_session.png`.

Ces fichiers ne doivent pas guider la nouvelle implémentation. Le contenu canonique sera créé dans
`content/` et les cours seront importés dans Moodle sous forme de paquets SCORM 1.2.

Les captures PNG sont volontairement ignorées par Git car elles peuvent contenir des informations
propres à l'environnement local.
